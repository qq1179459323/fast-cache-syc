    # fast-cache-ddza
    
    短小精悍的前端缓存工具，防止内存泄露

    特色（推广的一些亮点）
    
    
    ## 安装下载
    
    https://github.com/doudizhu/fast-cache-ddza/releases
    - `npm i fast-cache-npm`
    - CDN http://unpkg.com/fast-cache-ddza/release/bundle.js
    
    ## 快速使用
    
    ```
    var FC = window.FastCache;
    var cache = new FC();
    cache.set('a',100)
    alert(cache.get('a'))
    ```
